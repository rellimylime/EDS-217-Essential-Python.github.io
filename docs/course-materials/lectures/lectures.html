🎯 How to Use These Lectures

For Students

  1. Review before class: Read through the lecture materials to prepare
  2. Follow along: Use the presentations during live lectures
  3. Reference later: Return to specific topics when working on assignments
  4. Practice: Run code examples and modify them to learn

For Instructors

  1. Present in class: Use the reveal.js presentations for live lectures
  2. Customize: Modify content to fit your teaching style
  3. Share: Provide links to students for review
  4. Update: Keep content current with course developments

🚀 Presentation Features

All lectures are built using Quarto and reveal.js for optimal presentation experience:

  • Responsive design: Works on all devices and screen sizes
  • Interactive elements: Code execution and live demonstrations
  • Navigation: Easy slide navigation with keyboard shortcuts
  • Export options: Save as PDF or HTML for offline use
  • Accessibility: Screen reader friendly and keyboard navigable

📱 Keyboard Shortcuts

When viewing presentations: - Arrow keys or Spacebar: Navigate slides - ESC: Slide overview mode - F: Fullscreen mode - S: Speaker notes (if available) - ?: Show all keyboard shortcuts
