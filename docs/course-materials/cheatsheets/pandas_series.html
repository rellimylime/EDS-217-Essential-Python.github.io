Introduction

This cheatsheet provides a quick reference for common operations on Pandas Series. Itâ€™s designed for beginning data science students who are just starting to work with Pandas.

Importing Pandas

Always start by importing pandas:

Code 
import pandas as pd

Creating a Series

From a list

Code 
data = [1, 2, 3, 4, 5]
s = pd.Series(data)
print(s)
0    1
1    2
2    3
3    4
4    5
dtype: int64

From a dictionary

Code 
data = {'a': 0., 'b': 1., 'c': 2.}
s = pd.Series(data)
print(s)
a    0.0
b    1.0
c    2.0
dtype: float64

With custom index

Code 
s = pd.Series([1, 2, 3, 4, 5], index=['a', 'b', 'c', 'd', 'e'])
print(s)
a    1
b    2
c    3
d    4
e    5
dtype: int64

Selecting Data

By label

Code 
# Select a single element
element = s['a']

# Select multiple elements
subset = s[['a', 'c', 'e']]

By position

Code 
# Select by integer index
first_element = s[0]

# Select a slice
subset = s[1:4]
/var/folders/bs/x9tn9jz91cv6hb3q6p4djbmw0000gn/T/ipykernel_77376/1171660102.py:2: FutureWarning: Series.__getitem__ treating keys as positions is deprecated. In a future version, integer keys will always be treated as labels (consistent with DataFrame behavior). To access a value by position, use `ser.iloc[pos]`
  first_element = s[0]

By condition

Code 
# Select elements greater than 2
greater_than_two = s[s > 2]

Basic Data Manipulation

Updating values

Code 
s['a'] = 10

Applying functions

Code 
# Apply a function to each element
s_squared = s.apply(lambda x: x**2)

Handling missing values

Code 
# Drop missing values
s_cleaned = s.dropna()

# Fill missing values
s_filled = s.fillna(0)  # Fills with 0

Basic Calculations

Code 
# Calculate mean
mean_value = s.mean()

# Calculate sum
sum_value = s.sum()

# Calculate maximum
max_value = s.max()

Sorting

Code 
# Sort by values
s_sorted = s.sort_values()

# Sort by index
s_sorted_by_index = s.sort_index()

Reindexing

Code 
new_index = ['a', 'b', 'c', 'd', 'e', 'f']
s_reindexed = s.reindex(new_index)

Combining Series

Code 
s1 = pd.Series([1, 2, 3], index=['a', 'b', 'c'])
s2 = pd.Series([4, 5, 6], index=['d', 'e', 'f'])
s_combined = pd.concat([s1, s2])

Converting to Other Data Types

Code 
# To list
list_data = s.tolist()

# To dictionary
dict_data = s.to_dict()

# To DataFrame
df = s.to_frame()
